# Nmap Scan Lab

## Target
scanme.nmap.org

## Tool
Nmap

## Command Used

nmap -sS -sV -sC -p- -T4 scanme.nmap.org

## Purpose

This lab demonstrates network reconnaissance and service detection.

## Findings

Open ports discovered:
- 22 SSH
- 80 HTTP


## Skills Practiced

- Network scanning
- Service enumeration
- Reconnaissance
